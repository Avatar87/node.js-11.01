# Black Jack

reference rules / terms: https://www.pagat.com/banking/blackjack.html

## Assumptions:

- one player + dealer only;
- no money (yet)
- no split (yet)
- no insurance (yet)
- "five card charlie" rule: disabled

## How to start

1.
```
npm install
```

2. To play a game
```
npm start
```

3. To analyze your game logs
```
npm run logs
```

## Enjoy!
